# 🎈 AR Balloon Pop

AR Balloon Pop is an Augmented Reality (AR) mobile game developed using Unity and AR Foundation. The game allows players to interact with virtual balloons in an AR environment and pop them to increase their score before the timer ends.

## 📱 Project Overview

The application uses Augmented Reality to display virtual balloons in the user's real-world environment. Players can tap the balloons to pop them and earn points. A timer and score system make the game interactive and engaging.

## ✨ Features

- Augmented Reality gameplay
- Virtual balloon spawning
- Tap-to-pop interaction
- Score tracking system
- Countdown timer
- Game Over system
- Android mobile support

## 🛠️ Technologies Used

- Unity
- C#
- AR Foundation
- ARCore
- Android

## 🎮 How to Play

1. Install the APK on an AR-supported Android device.
2. Open the AR Balloon Pop application.
3. Allow camera permissions.
4. Move the phone to view the AR environment.
5. Tap the balloons to pop them.
6. Pop as many balloons as possible before the timer ends.
